# **App Name**: MediMind

## Core Features:

- Medication Management: Add and edit medications with details like name, dosage, schedule, and custom start/end dates.
- Intelligent Dose Scheduling: Continuously calculate upcoming doses and identify missed doses using a reusable scheduling service.
- Local Notifications: Schedule reliable local notifications for each upcoming dose with customizable channels, sounds, and actions (e.g., mark dose as taken).
- Home Dashboard: Display the next upcoming dose with a countdown, a timeline for today's doses, active medications, and a section for missed doses.
- Dose Logging: Allow users to mark scheduled doses as taken, skipped, or remind later; record each action in a dose history log.
- Inventory Tracking: Track medicine inventory (total, remaining quantity, refill threshold) and show smart alerts for low stock using generative AI tool
- Adherence Analytics: Visualize medication adherence over time using charts/stats with filters for day, week, month, and medicine.

## Style Guidelines:

- Primary color: Light blue (#90CAF9) for calmness and trustworthiness.
- Background color: Very light blue (#E3F2FD).
- Accent color: Slightly darker blue (#64B5F6) for interactive elements.
- Body and headline font: 'PT Sans' sans-serif for a balance of modernity and readability.
- Use clear, intuitive icons from Material Design to represent medication types and actions.
- Employ a clean, card-based layout for medication listings and dashboard elements.
- Subtle transitions and animations to provide feedback on user actions (e.g., marking a dose as taken).
